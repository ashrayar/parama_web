Help

1. Submitting a job

The user can submit a single protein structure in PDB format in the Home page, either by entering a valid PDB ID or by uploading a structure in PDB format (with .pdb as file extension). BoRaM will only work on one chain of the structure at a time and hence, chain ID must be entered. If not, the first chain in the structure will be taken as default. If it is a structure with multiple models, the first model will be considered for analysis.

The user can choose to either generate maps for all the non-proline residues in the structure or only specific residues. If the latter option is chosen, the preferred residue numbers must be entered as a comma separated list in the text field that appears upon choosing this option.

Note that the positions of backbone hydrogen atoms are required for the generation of Ramachandran maps. Hence please ensure that they are available in the PDB file. For residues which don't have the hydrogen atoms, maps will not be generated.

2. Viewing Results

If the protein structure that has been submitted is available in BoRaM database, the results will be immediately shown as a table, where each row corresponds to a residue and links to the bond-parameter specific map and the corresponding data file is provided. The table also indicates whether each residue is allowed or disallowed according to its Ramachandran map. If the structure is not present in database, the user will be asked to enter his/her email address, where the results will be sent. The time it takes for the results to be sent depends upon the number of residues for which maps have to be generated. Generating map for a single residue takes about 30 seconds.

This image is a sample map of a residue that will be generated as output. The green regions on the map denote the fully allowed conformations and off-white regions correspond to partially allowed conformations. The red point denotes the (ϕ-ψ) value in the protein structure for that residue. In this example, (ϕ-ψ) value in the crystal structure falls in partially allowed region of that residue's bond-parameter specific map.

The link to data corresponding to the generated map is also provided. This is a comma separated list file in which against every integral (ϕ-ψ) value, its conformational accessibility is indicated; 0 means disallowed, 1 means fully allowed and 2 means partially allowed.
